Stock Portfolio Tracker

Description

This is a simple Python project that helps users calculate the value of their stock portfolio. Users enter stock symbols and the number of shares they own. The program calculates the total investment value and stores the report in a text file.

Features

- Enter stock symbols and quantities
- Calculates portfolio value
- Generates a text report
- Uses Python file handling
- Beginner-friendly code

Technologies Used

- Python 3

How to Run

1. Install Python 3.
2. Save the file as "portfolio.py".
3. Open a terminal or command prompt.
4. Run:
   python portfolio.py
5. Enter stock details and view the generated report.
